# vue-mit-cli

![Npm 版本](https://img.shields.io/badge/mit-cli_v0.0.1-green)

## 📖 简介

- 用于快速搭建前端项目的命令行工具

## 📕 模板

- ✅ 快速生成 [Vue3+TS+Vite+ElementPlus+Pinia+Axios+Mock 的模板](https://gitee.com/Lpppsea9/admin-pro.git)

## 📦 安装

```bash
npm install mit-cli -g
```

## 🚩 使用

```bash
# 使用模板创建新项目
mit-cli create

# 执行项目文件夹名称并选择模板创建新项目
mit-cli create [projectName]

# 更新脚手架到最新版本
mit-cli update

# 查看脚手架版本
mit-cli -v
mit-cli --version

```
